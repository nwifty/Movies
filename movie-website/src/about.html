About Movie Wikipedia

Movie Wikipedia is a modern, community-driven encyclopedia for movies and TV shows. Our mission is to help users discover, search, and learn more about their favorite films and series in a visually appealing and easy-to-use interface.

  • Browse featured and top-rated movies
  • Search for movies by title, genre, or director
  • View detailed information about each movie
  • Enjoy a clean, modern design with responsive layouts

This website is a demo project for educational purposes, built with HTML, CSS, and JavaScript.
